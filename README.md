# Weather API Project ⛅

## How to run
- Is using laravel Sail
- Copy .env.example .env
- ./vendor/bin/sail up -d
- ./vendor/bin/sail migrate --seed

## Features ✨

- Migrations
- Factories
- Seeders
- FormRequests
- Sanctum
- Resources
- Http Client
- Gate, Policies

## Design 🏗️
- Service Repository pattern
- Factory Pattern for thirdy party Weather provider
